# Student Grade Calculator

## Overview
The **Student Grade Calculator** is a simple C++ console application that allows users to input a student's name and scores for three projects and two tests. The program calculates the student's average score and assigns a letter grade based on the average. The results are displayed neatly in the console.

---

## Features
- Input student **name** (supports full names with spaces).  
- Input scores for **3 projects** and **2 tests**.  
- Automatically calculates:
  - **Average score**
  - **Letter grade** (`A`, `B`, `C`, `D`, `F`)  
- Displays all student information in a clear format.  
- Includes a friendly **welcome message**.

---

## Requirements
- **Visual Studio** (Community, Professional, or Enterprise) or any C++ compiler that supports C++11 or later.  
- Basic knowledge of compiling and running C++ console applications.

---

## How to Run
1. Open **Visual Studio**.  
2. Open the solution or project containing `StudentGrades.cpp`.  
3. Build the project:
   - Go to **Build → Build Solution** or press `Ctrl + Shift + B`.  
4. Run the program:
   - Go to **Debug → Start Without Debugging** or press `Ctrl + F5`.  
5. Enter the requested information in the console when prompted
